# Blog de Café - Proyecto de Maquetación

Esta web esta dedicada al mundo del café, que incluye un blog con artículos, una sección de cursos y detalles sobre diferentes granos y preparaciones. El objetivo principal fue aplicar técnicas avanzadas de maquetación web y diseño visual.

## 🚀 Tecnologías Utilizadas

El sitio ha sido desarrollado con un enfoque en rendimiento y estándares web:

* **HTML5**: Marcado semántico para mejorar el SEO y la accesibilidad del contenido.
* **CSS3**: Implementación de layouts modernos (Flexbox y CSS Grid) para una distribución precisa de los elementos.
* **Google Fonts**: Uso de tipografías personalizadas para reforzar la estética rústica y profesional del blog.
* **Diseño Responsivo**: Adaptación completa de la interfaz para una lectura cómoda en dispositivos móviles y de escritorio.

## 🛠️ Características Principales

1. **Sección de Blog**: Maquetación de entradas con imágenes, fechas y descripciones breves.
2. **Cursos de Café**: Espacio dedicado a la promoción de talleres, utilizando componentes de diseño reutilizables.
3. **Optimización Visual**: Manejo eficiente de imágenes para garantizar una carga fluida sin perder calidad.
4. **Navegación Intuitiva**: Menú de navegación claro para facilitar el acceso a las diferentes secciones del sitio.

## 💻 Visualización Local

Para ver el proyecto en tu computadora, sigue estos pasos sencillos:

1. **Clonar el repositorio:**
```bash
git clone https://github.com/AndresPaqui/Blog-de-Cafe.git
```

2. **Entrar al directorio:**
```bash
cd Blog-de-Cafe
```

3. **Ejecutar el proyecto:**
Abre el archivo index.html en tu navegador favorito (Chrome, Firefox, Safari, etc.) para visualizar el sitio.

## 📂 Estructura del Proyecto

* **`index.html:`** Página de inicio del blog con las entradas principales.

* **`nosotros.html:`** Sección informativa sobre el blog o la cafetería.

* **`cursos.html:`** Catálogo de cursos disponibles.

* **`entrada.html:`** Plantilla para la visualización de un artículo individual.

* **`css/:`** Carpeta que contiene los estilos maestros del sitio.

* **`img/:`** Galería de imágenes utilizadas en el blog y los cursos.
